## How to

1. **Kör med .NET CLI**

```bash
dotnet run
```

2. **Ange start- och slut-IP**

```
Start IP: 192.168.50.0
End IP:   192.168.51.0
```

3. **Programmet visar alla IP-adresser som svarar**

```
192.168.50.1  - Active
192.168.50.5  - Active
...
```

## Tips

* Kör i terminal/kommandoprompt som admin.
* Stora nätverk kan ta lite tid att scanna.
